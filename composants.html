Système d'accordéons

Composant interactif pour FAQ, guides step-by-step, comparatifs

L'animation utilise max-height avec une transition CSS de 0.3s ease. L'icône chevron effectue une rotation de 90° lors de l'ouverture.

Le JavaScript calcule dynamiquement la hauteur du contenu pour une animation fluide.

Support complet du clavier :

  • Enter et Espace : Ouvrir/fermer
  • aria-expanded : État communiqué aux lecteurs d'écran
  • Focus visible : Outline bleu clair sur focus

Respecte prefers-reduced-motion pour désactiver les animations si nécessaire.

Pour ouvrir un accordéon par défaut, ajoutez la classe .accordion__item--open et définissez aria-expanded="true".

Le comportement par défaut ferme automatiquement les autres items (comportement exclusif).

Structure & classes

.accordion : Container principal
.accordion__item : Item individuel
.accordion__trigger : Bouton déclencheur
.accordion__content : Zone de contenu
.accordion__body : Contenu interne avec padding
.accordion__icon : Icône chevron (rotation automatique)

JavaScript requis : initAccordions()
